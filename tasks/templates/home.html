Inicio

Aplicación web de Gestion de Tareas para crear, editar y organizar tareas. Se incluyen el registro e inicio de sesión, tareas prioritarias, marcado de tareas completadas y vistas filtradas (pendientes / completadas). Interfaz responsiva con Bootstrap y almacenamiento en SQLite.

  • Registro e inicio de sesión por usuario
  • Crear, editar y eliminar tareas
  • Marcar tareas como importantes
  • Marcar tareas como completadas y ver historial
  • Vistas filtradas y protección de rutas (login requerido)
Arquitectura y patrones

Esta aplicación sigue la arquitectura Django (MTV: Models, Templates, Views) con separación clara de capas:

  • Backend: Django 5 (vistas basadas en funciones), Django ORM para persistencia en SQLite, autenticación basada en sesiones.
  • Models: lógica de datos y mapeo a la base (patrón Active Record a través del ORM).
  • Views/Controllers: funciones que gestionan solicitudes HTTP, validación con Django Forms y uso de decoradores como login_required para protección de rutas.
  • Templates / Frontend: Django Templates + Bootstrap para UI responsiva; uso de template inheritance (base.html) para mantener DRY.
  • Patrones aplicados: separación de responsabilidades (SoC), uso de nombres de rutas y reversión para desacoplar enlaces, manejo de formularios (Form objects) y protección CSRF.

Tecnologías: Python, Django, HTML, CSS, JavaScript, Bootstrap, SQLite.

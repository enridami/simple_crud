Inicio

Aplicación gestión de tareas realizado en el lenguaje Python, con el framework Django, HTML, CSS (Bootstrap), con motor de base de datos SQLite.

La aplicación permite registrar usuario, inciar sesión y realizar la gestión de tareas:

  • Crear tarea
  • Modificar tarea
  • Determinar cual tarea es la importante dentro de la lista
  • Marcar tarea completada
  • Listar tareas pendientes
  • Listar tareas completadas
